## 한 일

### LoginPage (A-1)

- `useState` 로 pin/submitting/error 관리.
- 입력 처리: `value.replace(/\D/g, '')` — 숫자만 통과, maxLength=6.
- 정규식 검증: `/^\d{6}$/` — 6자리 만족 시에만 submit 진행.
- `apiFetch(API.ADMIN_LOGIN, { method: 'POST', body: { pin } })`.
- 401 → "PIN이 일치하지 않습니다" 인라인 (토스트 X — UX-1).
- 그 외 ApiError → `err.message` 인라인.
- 성공 → `navigate('/admin/dashboard')`.
- atoms: `Label` (필수 별표) + `Input` (invalid·errorMessage) + `Button` (block, loading, disabled).

### OrderDetailPage (A-3)

- `useApi(({ signal }) => apiFetch(API.ADMIN_ORDER(id), { schema: OrderSchema, signal }))`.
- 분기: `isLoading` → LoadingState page / `error.status === 401` → `/admin/login` redirect / 그 외 error → ErrorState + 재시도.
- 6 액션 매핑 상수 `ACTIONS` (모듈 최상위):

| key | 라벨 | to | from |
|---|---|---|---|
| confirm_transfer | 이체 확인 | PAID | TRANSFER_REPORTED |
| hold | 보류 | HOLD | TRANSFER_REPORTED |
| cancel | 취소 | CANCELED | ORDERED·TRANSFER_REPORTED·PAID·HOLD |
| start_cooking | 조리 시작 | COOKING | PAID |
| finish_cooking | 조리 완료 | READY | COOKING |
| deliver | 전달 완료 | DONE | READY |

- `availableActions = ACTIONS.filter(a => a.from.includes(order.status))` — UI 가용성. **합법 전이 최종 검증은 백엔드** (ADR-025).
- 액션 클릭 → `apiFetch(API.ADMIN_ORDER_TRANSITION(id), { method:'POST', body:{ action, to } })` → `orderQuery.refetch()`.
- 액션 401 → `/admin/login` redirect / 그 외 ApiError → 인라인 `actionError` (토스트 X).
- 표시 요소: 헤더(주문 번호 + StatusChip) · 주문 항목 리스트 · 합계 · OrderTimeline (history miniview) · 액션 버튼 그리드.
- cancel/hold 만 `variant="danger"`, 나머지 `primary`.

### TransfersPage (A-4)

- zod `TransferListSchema` (id·no·depositor_name·bank·amount·transferred_at·status — array).
- `useApi(... { schema: TransferListSchema, signal })`.
- 분기: Loading → LoadingState / 401 → `/admin/login` redirect / error → ErrorState / 빈 배열 → EmptyState ("확인 대기 중인 이체가 없어요") / 목록 → 카드 ul.
- 모든 분기 공통 `<Wrapper data-testid="admin-transfers-page">` — App.test.jsx 라우팅 회귀 보호.
- 카드 클릭 → `navigate('/admin/orders/${t.id}')`.
- 안전 가드 `Array.isArray(query.data) ? query.data : []` — 스키마 모킹 우회 케이스도 EmptyState 분기.
