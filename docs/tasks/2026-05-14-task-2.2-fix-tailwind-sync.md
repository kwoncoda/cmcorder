## 한 일

### 1. `tailwind.config.js` extend 보강

- `theme.extend.fontSize` 10종 추가 — `3xs` · `2xs` · `xs` · `sm` · `base` · `lg` · `xl` · `2xl` · `3xl` · `4xl` 모두 `var(--text-*)` 토큰으로 위임.
- `theme.extend.borderRadius` 에 `none` · `pill` 추가 (`var(--radius-none)` · `var(--radius-pill)`).
- 이로써 tokens.css 의 모든 typography · radius 토큰이 빌드 CSS 에 실제 클래스로 생성됨.

### 2. `MenuFallback.jsx` SIZE_CLASSES md/lg 교환

```diff
-  sm: 'text-2xl',
-  md: 'text-4xl',
-  lg: 'text-3xl',
+  sm: 'text-2xl', // 32px — 리스트 small
+  md: 'text-3xl', // 48px — 일반 메뉴 카드
+  lg: 'text-4xl', // 64px — 강조 (상세·CTA)
```

명시 사이즈 테스트가 `MenuFallback.test.jsx` 에 없어 안전.

### 3. `IconLabel.jsx` 코멘트 정정

`gap: 'xs' (2px), 'sm' (8px), 'md' (12px)` → `gap: 'xs' (4px = --space-2xs), 'sm' (8px = --space-xs), 'md' (12px = --space-sm)`.

### 4. 회귀 테스트 신설 — `src/__tests__/tokens-tailwind-sync.test.js`

`tokens.css` 의 `--text-*` · `--space-*` · `--radius-*` 변수를 정규식으로 추출해 각각 `tailwind.config.js` 의 `var(--TOKEN)` 매핑 존재를 회귀 검사. 향후 토큰 추가 시 tailwind.config 누락이 즉시 적색 신호.
