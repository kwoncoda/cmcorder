### 2. GREEN — 컴포넌트 구현

**핵심 패턴 (§3.5 4조 — rerender-derived-state-no-effect):**

```jsx
const [shouldAnimate] = useState(() => {
  if (typeof window === 'undefined') return false;
  if (!dropping || no == null) return false;
  const key = `dogtag-shown-${no}`;
  if (sessionStorage.getItem(key)) return false;
  sessionStorage.setItem(key, '1');
  return true;
});
```

- `useState 초기화 함수`는 첫 렌더 *직전* 1회 실행 → 첫 페인트부터 `dogtag-drop` 클래스 포함.
- StrictMode 2-mount 환경에서도 초기화 함수는 한 번만 실행 → sessionStorage 키도 1회.
- 시안(docs/design-bundle/components.jsx line 48-75)의 `useEffect` + `useRef` 조합은 폐기.

**진동 API (mount 1회 부수효과):**

```jsx
useEffect(() => {
  if (!shouldAnimate) return;
  const mq = window.matchMedia?.('(prefers-reduced-motion: reduce)');
  if (mq?.matches) return;
  navigator.vibrate?.([60, 30, 60]);
}, [shouldAnimate]);
```

- `shouldAnimate` 게이트가 false 면 진동 X (재방문 / dropping=false).
- reduced motion 사용자 회피 — `matchMedia` optional chaining (SSR/jsdom 안전).
- Safari 미지원 → `navigator.vibrate?.()` optional chaining.

**외형 (DESIGN §1 + §5.1):**

- bg-card-bg (흙색) + text-card-ink + rounded-tag (`8px 8px 4px 4px`) + shadow-elevated
- 주문번호: `font-display font-black text-3xl`
- 총수: `text-sm opacity-60`
- 일자: `font-mono tabular-nums text-xs opacity-70`
- ORDER NO 레이블: `text-3xs uppercase tracking-wider opacity-70`
- origin-top — 떨어지는 모션 회전축 (자연스러운 진자 운동)

### 3. CSS keyframes (`src/styles/components.css`)

별도 파일로 분리 — globals.css 의 reset/접근성과 분리. main.jsx 에서 globals.css 다음 import.

```css
@keyframes dogtag-drop {
  0%   { transform: translateY(-180px) rotate(-10deg); opacity: 0; }
  50%  { transform: translateY(20px) rotate(2deg); opacity: 1; }
  75%  { transform: translateY(-8px) rotate(-1deg); }
  100% { transform: translateY(0) rotate(0); opacity: 1; }
}
.dogtag-drop { animation: dogtag-drop var(--duration-tag) var(--ease-tag) both; }

@keyframes dogtag-pulse-yellow {
  0%, 100% { box-shadow: var(--shadow-elevated); }
  50%      { box-shadow: var(--shadow-elevated), 0 0 32px var(--color-accent); }
}
.dogtag-pulse { animation: dogtag-pulse-yellow 1.2s ease-in-out infinite; }

@media (prefers-reduced-motion: reduce) {
  .dogtag-drop, .dogtag-pulse { animation: none; }
}
```

`--duration-tag` (600ms) + `--ease-tag` (`cubic-bezier(0.17, 0.67, 0.32, 1.5)`) 는
tokens.css 정의 사용.
