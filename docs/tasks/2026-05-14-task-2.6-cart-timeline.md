### 2. GREEN — CartItem.jsx 구현

핵심:

```jsx
const CartItem = forwardRef(function CartItem(
  { menu, quantity = 1, onChangeQty, onRemove, useFallback = true, ... },
  ref,
) {
  if (!menu) return null;
  const subtotal = (menu.basePrice ?? 0) * quantity;

  const handleDec = () => {
    const next = quantity - 1;
    if (next <= 0) onRemove?.(menu.id);   // ★ 0 도달 시 자동 remove
    else onChangeQty?.(menu.id, next);
  };
  const handleInc = () => onChangeQty?.(menu.id, quantity + 1);

  // <article aria-label="인벤토리 항목 — {name}">
  //   <div>이미지/이모지 (MenuFallback 또는 <img>)</div>
  //   <div>본명 h3 + 단가 PriceTag + 소계 PriceTag (text-card-ink)</div>
  //   <div>[- Button] <span aria-live="polite">qty</span> [+ Button]</div>
});
```

설계 결정:

- **수량 1에서 − 자동 remove**: 부모는 항상 양수 quantity 만 처리 → 데이터 모델 단순화 (0/null 분기 없음).
- **aria-live="polite"**: assertive 가 아닌 polite — 사용자 입력 직접 결과라 강제 끼어들기 X.
- **카드 톤 `bg-card-bg text-card-ink`**: MenuCard 와 동일 흙색 톤. 형광 옐로 X (AI 슬롭 #26).
- **단가·소계 모두 PriceTag**: 천 단위 콤마 (`Intl.NumberFormat('ko-KR')`) 일관.

const STEPS = [   // 모듈 최상위 상수 (SoT)

const OrderTimeline = forwardRef(function OrderTimeline(

  else {

    doneCount = idx < 0 ? 0 : idx;                        // 알 수 없는 값 → 0
