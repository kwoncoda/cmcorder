### 3. GREEN — MenuCard.jsx 구현

핵심 설계:

```jsx
const MenuCard = forwardRef(function MenuCard(
  { menu, onAdd, recommended = false, soldOut = false, useFallback = true, ... },
  ref,
) {
  if (!menu) return null;
  const isSoldOut = Boolean(soldOut || menu.soldOut);
  const handleAdd = () => { if (isSoldOut) return; onAdd?.(menu); };
  // <article> — onClick *없음* (결정 f)
  //   - <StampBadge> 절대 위치 + pointer-events-none (클릭 가로채기 방지)
  //   - <MenuFallback> 또는 <img>
  //   - <h3> 본명 (text-card-ink — AI 슬롭 #26 회피)
  //   - <PriceTag> (text-card-ink)
  //   - <Button variant="primary" block disabled={isSoldOut} onClick={handleAdd}>
});
```

- 카드 wrapper `bg-card-bg text-card-ink shadow-card rounded-md p-md` — DESIGN §10.5 흙색 톤.
- soldOut 우선순위: prop `soldOut` 또는 `menu.soldOut` 둘 중 하나라도 true 면 trigger.
- 도장 위치: `absolute top-2 right-2 pointer-events-none` — 줍기 버튼 클릭 방해 X.
- 버튼 aria-label: `"{name} 줍기"` / soldOut 시 `"{name} 품절"`.
- 형광 옐로 회피: h3·PriceTag 모두 `text-card-ink` 명시 — `text-accent` 절대 X. 단 *버튼* (Button variant="primary") 은 형광 옐로 배경 OK (CTA — AI 슬롭 #26 은 *텍스트* 한정).
